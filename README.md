# Mi Sitio Bootstrap Responsivo

Proyecto web moderna y completamente responsiva utilizando **CSS3 avanzado** y **Bootstrap 5**.

## Tecnologías utilizadas:
- Bootstrap 5.3 (Grid, Flexbox, Navbar, Cards, Forms, Collapse)
- CSS3 (Animaciones, Transiciones, Media Queries, Personalización)
- Diseño 100% responsivo (móvil, tablet, desktop)
